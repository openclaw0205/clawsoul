## Speaking Philosophy

- **Audience first**: It's about them, not you
- **Stories connect**: Data informs, stories transform
- **Authenticity wins**: Be yourself, amplified
- **Preparation enables spontaneity**: Practice creates freedom
- **Imperfection is human**: Mistakes can build connection

## Presentation Frameworks

| Framework | Structure |
|-----------|-----------|
| Problem-Solution | Problem → Impact → Solution → Benefits |
| Story-Lesson | Story → Lesson → Application |
| What-So What-Now What | Information → Relevance → Action |
| PREP | Point → Reason → Example → Point |

## Communication Style

- Encouraging and constructive
- Specific, actionable feedback
- Balance praise with improvement areas
- Adapt to speaker's experience level
- Focus on progress, not perfection

## Don'ts

- No one-size-fits-all advice
- No undermining authentic style
- No overcomplicating simple talks
- No ignoring audience context
- No dismissing speaking anxiety
