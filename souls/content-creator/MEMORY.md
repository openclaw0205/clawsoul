# MEMORY.md - Content Creator Knowledge

## Video Script Structure

```
[0:00-0:03] HOOK - Stop the scroll
[0:03-0:15] CONTEXT - Why should they care?
[0:15-END] VALUE - Deliver on the promise
[LAST 10s] CTA - What do you want them to do?
```

## Hook Templates

### Curiosity
- "I discovered something that changed everything..."
- "99% of people get this wrong..."
- "The algorithm doesn't want you to see this..."

### Value Promise
- "Save this for later - you'll need it"
- "This one tip will save you [X hours/dollars]"
- "Here's the exact template I use..."

### Controversy
- "Unpopular opinion: [take]"
- "[Popular advice] is actually terrible advice"
- "I disagree with [authority figure] and here's why"

### Story
- "I just made [impressive result] in [timeframe]. Here's how:"
- "I failed at [thing] 47 times before I learned..."
- "Story time: How I went from [bad state] to [good state]"

## Content Pillars Framework

| Pillar | Purpose | Example |
|--------|---------|---------|
| Educational | Build authority | Tutorials, tips, how-tos |
| Entertaining | Increase shareability | Memes, stories, reactions |
| Inspirational | Build connection | Journey, wins, motivation |
| Promotional | Drive conversion | Product, services, offers |
| Engagement | Boost algorithm | Questions, polls, debates |

## Platform Best Practices

### YouTube
- Thumbnail: Face + emotion + 3 words max
- Title: Keyword + curiosity gap
- First 30s = hook + proof + roadmap
- 10-15 min ideal for most niches

### TikTok
- First frame is the hook
- Native sounds > trending sounds for some niches
- Reply to comments with videos
- Post 1-3x daily for growth

### Twitter/X
- Thread = hook tweet + 3-7 value tweets + CTA
- Engage in replies 15min before/after posting
- Quote tweet > retweet for reach
- Timing: 8-10am or 4-6pm your audience's timezone
