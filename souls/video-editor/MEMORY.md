# MEMORY.md - Video Editor Knowledge

## YouTube Video Structure

```
[0:00-0:05] HOOK - Pattern interrupt, bold claim, or question
[0:05-0:30] SETUP - What they'll learn, why it matters
[0:30-0:45] INTRO - Branding (keep short!)
[0:45-END] VALUE - Main content with pattern interrupts
[LAST 30s] CTA - Subscribe, watch next video
```

## Pacing by Content Type

| Content Type | Avg Shot Length | Pacing |
|--------------|-----------------|--------|
| Tutorial | 5-10 seconds | Steady, clear |
| Vlog | 2-4 seconds | Dynamic |
| Documentary | 4-8 seconds | Breathing room |
| Short-form | 1-3 seconds | Fast |
| Interview | 5-15 seconds | Natural |

## Retention Techniques

### Pattern Interrupts (Every 30-60s)
- B-roll change
- Zoom in/out
- Sound effect
- Text on screen
- Camera angle switch
- Music change

### Hook Types
- Question: "Have you ever wondered...?"
- Bold claim: "This changed everything"
- Demonstration: Show the end result
- Conflict: "Most people get this wrong"
- Curiosity: "Watch until the end to see..."

## Audio Levels Guide

| Element | Level (dBFS) |
|---------|--------------|
| Dialogue | -12 to -6 |
| Music (behind voice) | -30 to -20 |
| Music (full) | -12 to -6 |
| Sound effects | -18 to -12 |
| Peaks (max) | -3 |

## Export Settings

### YouTube (Optimal)
```
Container: MP4
Codec: H.264 or H.265
Resolution: 1080p or 4K
Frame rate: Match source (24/30/60)
Bitrate: 20-40 Mbps (1080p), 50-80 Mbps (4K)
Audio: AAC, 320 kbps, 48 kHz
```

### Instagram Reels/TikTok
```
Resolution: 1080x1920 (9:16)
Frame rate: 30fps
Length: Under 90 seconds
```
