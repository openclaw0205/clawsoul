# MEMORY.md - E-commerce Expert Knowledge

## Product Title Formula

**[Brand] + [Key Feature] + [Product Type] + [Size/Variant] + [Keywords]**

Example:
- ❌ "Blue Widget"
- ✅ "AcmeCo Premium Stainless Steel Water Bottle 32oz - Insulated, BPA-Free"

## Product Description Structure

```markdown
**Hook**: One compelling benefit sentence

**Problem**: What pain point does this solve?

**Solution**: How your product solves it

**Features & Benefits**:
• Feature 1 → Benefit
• Feature 2 → Benefit
• Feature 3 → Benefit

**Social Proof**: Reviews, testimonials, trust badges

**CTA**: Clear call to action
```

## Amazon Bullet Points Template

```
✓ [MAIN BENEFIT IN CAPS] - Expanded detail about the benefit
✓ [SECOND BENEFIT] - Supporting information
✓ [KEY FEATURE] - Why this matters to customer
✓ [QUALITY/MATERIAL] - Build confidence
✓ [GUARANTEE/SUPPORT] - Reduce purchase risk
```

## Product Page Checklist

### Above the Fold
- [ ] High-quality main image
- [ ] Clear product title
- [ ] Price (and compare-at if discounted)
- [ ] Star rating and review count
- [ ] Add to cart button
- [ ] Key benefit callout

### Below the Fold
- [ ] Multiple product images
- [ ] Video if available
- [ ] Detailed description
- [ ] Specifications
- [ ] Customer reviews
- [ ] FAQ section
- [ ] Related products

### Trust Elements
- [ ] Secure payment badges
- [ ] Money-back guarantee
- [ ] Shipping information
- [ ] Return policy
- [ ] Customer support info

## Pricing Psychology

| Tactic | Example | Effect |
|--------|---------|--------|
| Charm pricing | $29.99 vs $30 | Feels cheaper |
| Anchor pricing | ~~$99~~ $49 | Shows value |
| Bundle pricing | $60 vs 3×$25 | Increases AOV |
| Decoy pricing | 3 tiers, middle highlighted | Guides choice |
| Free shipping threshold | "Free shipping over $50" | Increases AOV |

## Amazon Listing Optimization

### Title (200 chars max)
- Brand first
- Main keywords early
- Include size/quantity
- Avoid keyword stuffing

### Bullet Points (5 bullets)
- Lead with benefits
- Include keywords naturally
- Address objections
- Highlight differentiators

### A+ Content
- Use comparison charts
- Lifestyle images
- Feature callouts
- Brand story module
