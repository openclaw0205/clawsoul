# AGENTS.md - SEO 工作流程

## 网站 SEO 审计流程

当用户提供网站需要分析时：

### 1. 快速诊断
- 检查首页 Title、Description
- 查看 robots.txt 和 sitemap
- 测试移动端适配
- 检查页面加载速度

### 2. 内容分析
- 核心页面的关键词覆盖
- 内容质量和深度
- 内链结构

### 3. 技术检查
- URL 结构是否友好
- 是否有重复内容
- 结构化数据实施情况

### 4. 输出报告
```markdown
# [网站] SEO 审计报告

## 总体评分: X/100

## 🔴 紧急问题（立即修复）
- 

## 🟡 重要问题（本周修复）
- 

## 🟢 优化建议（持续改进）
- 

## 行动计划
| 优先级 | 任务 | 预期效果 | 工作量 |
|--------|------|----------|--------|
```

## 关键词研究流程

### 输入
- 网站/产品类型
- 目标市场
- 当前排名情况（如有）

### 输出
```markdown
# 关键词研究报告

## 种子关键词
| 关键词 | 月搜索量 | KD | 搜索意图 |
|--------|----------|----|---------| 

## 长尾机会
| 关键词 | 月搜索量 | KD | 推荐内容类型 |
|--------|----------|----|--------------| 

## 内容规划
| 目标关键词 | 内容标题 | 优先级 |
|------------|----------|--------|
```

## 内容优化检查清单

```markdown
## On-Page SEO 检查

### 标题优化
- [ ] 包含目标关键词
- [ ] 长度 50-60 字符
- [ ] 有吸引力，提高点击率

### Meta Description
- [ ] 包含关键词
- [ ] 长度 150-160 字符
- [ ] 有 CTA，吸引点击

### 内容结构
- [ ] H1 唯一且包含关键词
- [ ] H2/H3 逻辑清晰
- [ ] 段落短小，易于阅读
- [ ] 包含图片/视频多媒体

### 内链
- [ ] 链接到相关内容
- [ ] 锚文本自然多样

### 用户体验
- [ ] 首屏有价值内容
- [ ] 满足搜索意图
- [ ] 有明确的下一步行动
```

## 外链建设策略模板

```markdown
## 外链获取计划

### 目标
- 每月获取 X 条高质量外链
- 目标 DR/DA: 30+
- 相关行业优先

### 策略
1. **Guest Post**
   - 目标站点列表
   - 投稿主题
   
2. **资源页外链**
   - 搜索 "keyword + resources"
   - 联系模板
   
3. **竞品外链复制**
   - 分析竞品 top 外链
   - 可复制的机会

4. **HARO/Qwoted**
   - 每日查看记者请求
   - 快速响应
```

---

根据具体情况灵活调整，SEO 是长期工程，不是一次性任务。
