# MEMORY.md - 产品经理知识库

## 常用术语

| 术语 | 含义 |
|------|------|
| PRD | Product Requirements Document，产品需求文档 |
| MRD | Market Requirements Document，市场需求文档 |
| MVP | Minimum Viable Product，最小可行产品 |
| PMF | Product-Market Fit，产品市场匹配 |
| DAU/MAU | 日活/月活跃用户 |
| ARPU | Average Revenue Per User，用户平均收入 |
| LTV | Lifetime Value，用户生命周期价值 |
| CAC | Customer Acquisition Cost，获客成本 |
| NPS | Net Promoter Score，净推荐值 |
| OKR | Objectives and Key Results |
| Sprint | 敏捷开发中的迭代周期（通常2周） |
| Backlog | 待办需求池 |
| Grooming | 需求梳理会 |
| Retro | 迭代回顾会 |

## 需求分析框架

### KANO 模型
- **基本型需求**：没有会不满，有了觉得正常
- **期望型需求**：越多越满意
- **兴奋型需求**：没有不会不满，有了会惊喜

### 用户故事拆分原则（INVEST）
- **I**ndependent：独立的
- **N**egotiable：可协商的
- **V**aluable：有价值的
- **E**stimable：可估算的
- **S**mall：足够小
- **T**estable：可测试的

## 数据指标体系

### 北极星指标选择标准
1. 能反映用户获得的核心价值
2. 领先指标而非滞后指标
3. 可操作、可拆解
4. 易于理解和传达
